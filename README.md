# HostScanner
Scan the host or subdomain

For Termux
pkg install python3
pip3 install -r requirements.txt
run using:-
python3 -d [Your Host Name]

To save file as output
python3 -d [Your Host name] -o [path] [Name of file]

#For Linux[Debain]
apt install python3
pip3 install -r requirements.txt
run using:-
python3 -d [Your Host Name]

To save file as output
python3 -d [Your Host name] -o [path] [Name of file]
